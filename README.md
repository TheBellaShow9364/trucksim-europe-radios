**TruckSim Europe Radios** est une collection de stations radio françaises pour :

👉 Euro Truck Simulator 2        
👉 American Truck Simulator 

🎧 Profite de **100 radios françaises prêtes à l’emploi** pour une immersion totale.

## 📥 Téléchargement

👉 **Version actuelle : v0.1 (bêta)**  
Télécharger dans les releases
---

## 🛠️ Installation
1. Télécharger le fichier  
2. Ouvrir le dossier Documents  
3. Aller dans : Euro Truck Simulator 2  ou American Truck Simulator  
4. Copier le fichier dans ce dossier  
5. Lancer le jeu  
6. Appuyer sur la touche R pour accéder aux radios
